# Description

This is a E-commerce website with functions:
+ View, search product(book)
+ Add product to cart
+ Create an order
+ Booking
+ Payment with Paypal
+ Order tracking
+ Api for that functions

# Use:
- Mysql
- Spring boot
- Spring security
- Thymleaf
- Bootstrap
- Restful
- Tomcat
